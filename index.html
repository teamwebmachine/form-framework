Form-Framework

A form framework based on bootstrap and uses gulp to compile less, scss and javascript.

View Example View on GitHub Download .tar.gz

These forms are on point.

The Form-Framework is a starter kit to create great webs forms and includes: gulp, bootstrap, sass & less, formvalidation.io, iframe.Resizer and much more. Installation is super simply:

$ git clone https://github.com/teamwebmachine/form-framework.git
$ sudo npm install
$ gulp watch
Form-Framework is maintained by teamwebmachine. Hosted by Github Pages.
