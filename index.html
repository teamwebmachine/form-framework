Form-Framework

A form framework based on bootstrap and uses gulp to compile less, scss and javascript.

View Example View on GitHub Download .tar.gz

These forms are on point.

The Form-Framework is a starter kit to create great webs forms and includes: gulp, bootstrap, sass & less, formvalidation.io, iframe.Resizer and much more. Installation is super simp:

$ git clone https://github.com/teamwebmachine/form-framework.git
$ sudo npm install
$ gulp watch

Bower packages are included in the repo, but feel free to delete the assets/vendor folder and use bower install to gather bower packages.

Dependencies

  • The form uses PHP, but you can easily convert to static html.
  • Node.js & NPM.
  • See the Package.json for required npm modules.
